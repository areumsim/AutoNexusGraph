# 프로젝트 운영 규칙 (Claude 전용)

## 절대 건들지 말 것 — 사용자 개인 작업 파일

다음 파일은 **사용자의 개인 작업 메모/스크래치패드** 입니다. Claude 는 절대로
읽거나, 쓰거나, 수정하거나, 삭제하지 마십시오. `.gitignore` 에 등록되어 있지만
로컬 파일은 사용자의 것입니다.

- `_tmp.txt`
- `_api_tmp.txt`
- 그 외 `_*tmp*` 패턴 매칭되는 모든 파일

위 파일에 대해서는:
- `Read` — 절대 금지 (내용 자체를 보지 마세요)
- `Write`, `Edit`, `NotebookEdit` — 절대 금지 (수정/덮어쓰기 금지)
- `Bash` 의 `rm`, `mv`, `cp`, `sed`, `truncate` 등 — 절대 금지
- "정리" / "보안" / "임시 파일이니까" 같은 임의 판단으로 손대지 마세요

`settings.local.json` 의 `permissions.deny` 에 hard-block 도 동시 등록되어
있어, 실수해도 차단됩니다.

**사유**: 2026-05-29 세션에서 Claude 가 사용자 명시 동의 없이 위 파일들을 삭제
하여 사용자 작업 메모를 손실시킨 사고가 있었습니다. 동일 사고 재발 방지.

## 일반 원칙

- 사용자가 명시적으로 요청하지 않은 행위는 하지 마십시오.
- 옵션을 선택지로 제시한 경우, 후속 질문에서 "진행 안 함" 이 나오면 그 옵션의
  실행도 함께 보류해야 합니다.
